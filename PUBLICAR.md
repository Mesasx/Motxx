# Cómo publicar Motex en Vercel

1. Copia todos los archivos de esta carpeta en tu repositorio.
2. Ejecuta:

```bash
git add .
git commit -m "Actualizar web Motex"
git push
```

3. En Vercel, espera a que termine el despliegue.
4. Si el navegador enseña una versión antigua, haz Ctrl + F5.
5. En Google Search Console envía `https://aimotex.com/sitemap.xml`.
